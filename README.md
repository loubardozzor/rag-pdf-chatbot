## 🧠 What is this app ?

**RAG PDF Chatbot** is an AI-powered application that allows users to upload any PDF document and ask questions about its content in natural language.

Instead of reading through long documents manually, the user simply uploads a PDF and chats with it — the app retrieves the most relevant sections and generates accurate, context-aware answers using GPT-4.

---

## ✨ Features

- 📁 Upload any PDF document
- 💬 Ask questions in natural language
- 🔍 Semantic search powered by vector embeddings
- 🤖 Accurate answers generated by GPT-4
- ⚡ Smart caching — embeddings are computed only once per document

---

## 🏗️ Architecture

The application is built around the **RAG (Retrieval-Augmented Generation)** pattern :

```
PDF Document
     │
     ▼
Text Extraction (PyPDFLoader)
     │
     ▼
Text Chunking (RecursiveCharacterTextSplitter)
     │
     ▼
Embedding Generation (OpenAIEmbeddings)
     │
     ▼
Vector Storage (ChromaDB)
     │
     ▼
User Query ──► Embedding ──► Similarity Search
                                    │
                                    ▼
                             Relevant Chunks
                                    │
                                    ▼
                        Prompt Template (System + User)
                                    │
                                    ▼
                              GPT-4 (ChatOpenAI)
                                    │
                                    ▼
                             Final Answer
```

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/rag-pdf-chatbot.git
cd rag-pdf-chatbot
```

### 2. Create a virtual environment

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Set up your API key

Create a `.env` file in the root directory :

```
OPENAI_API_KEY=your-openai-api-key-here
```

### 5. Run the app

```bash
streamlit run app.py
```

The app will open automatically in your browser at `http://localhost:8501`.
